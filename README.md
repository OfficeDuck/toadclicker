# toadclicker
an auto clicker made for minecraft. 

***Features***

**Clicker Tab**

- Left Autoclicker
- Rmb lock
- Blatant Mode
- Slot Whitelist
- Click in inventory
- Prioritize higher cps
- Slot whitelist
- Advanced settings: (custom randomazation, a graph, adjust spikes and boosts)

- Right Autoclicker
- Click in inventory
- Click only in inventory

**Configs Tab**
- Presets for popular servers
- Custom config system

**Recorder Tab**
- Settings and info box
- Record click delays and play them back
- Save and load system
- See average cps, record status, total clicks and delays. 

**Misc Tab**
- Beep on toggle
- Hide bind
- Custom Window options
- Jitter
- Set color themes
