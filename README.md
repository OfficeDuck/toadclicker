# toadclicker
an auto clicker made for minecraft. 

***Features***

**Clicker Tab**

- Left Autoclicker
- rmb lock
- Blatant Mode
- Slot Whitelist
- click in inventory
- prioritize higher cps
- slot whitelist

- Right Autoclicker
- click in inventory

**Misc**
- Beep on toggle
- Hide bind
- Jitter
