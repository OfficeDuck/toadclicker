# toadclicker
an auto clicker made for minecraft. 

***Features***

**Clicker Tab**

- Left Autoclicker
- Rmb lock
- Blatant Mode
- Slot Whitelist
- Click in inventory
- Prioritize higher cps
- Slot whitelist

- Right Autoclicker
- Click in inventory
- Click only in inventory

**Misc Tab**
- Beep on toggle
- Hide bind
- Custom Window options
- Jitter
